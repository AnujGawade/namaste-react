# Namaste React 🚀

This repository documents my learning journey through the **Namaste React Course by Akshay Saini**.
It contains episode-wise code implementations, notes, assignments, and small projects built while understanding the fundamentals and internals of React.

The goal of this repository is to strengthen my understanding of React concepts and build production-level frontend applications.

---

## 📚 About the Course

The Namaste React course focuses on building a strong foundation in React by understanding:

* React fundamentals
* JSX and React.createElement
* Functional Components
* Props and State
* React Hooks
* Virtual DOM and Reconciliation
* Event Handling
* Routing
* Performance Optimization
* Real-world React application development
